# Introduction
Σκοπός της συγκεκριμένης εργασίας είναι η δημιουργία ενός 2D παιχνιδιού. Θα πρέπει να δημιουργηθεί μια πίστα παιχνιδιού με χαρακτήρες, εχθρούς, αντικείμενα, εφέ, ήχο, κίνηση κτλ. της επιλογής μας. # Summary

# Conclusions
Τελικό συμπέρασμα, έχει δημιουργηθεί ένα λειτουργικό top-down RPG παιχνίδι τύπου Ruby's Adventure με βάση τα ξεχωριστά ζητούμενα του κάθε παραδοτέου. Έχουν γίνει διάφορες αλλαγές προκειμένου να συμβαδίζει το παιχνίδι με τη θεματολογία και τον σχειασμό που έχω ακολουθήσει. Αποτελεί ένα παιχνίδι με όνομα "Riri the cat" που η γάτα είναι ο βασικός πρωταγωνιστής του παιχνιδιού. Μέσα από την συλλογή κοκάλων και κολοκυθών, ενισχύεται η ζωή της γάτας. Στόχος είναι να αποφεύγει τις νεκροκεφαλές καθώς και τον εχθρό που την κάνουν να μειώνει τη ζωή της. Η γάτα πρέπει να πυροβολέι τον εχθρό μέχρι να τον σκοτώσει για να μειώσει τις πιθανότητες να χάσει ζωή. Υπάρχουν 2 ταφόπλακες που βοηθούν την τηλεμεταφορά της γάτας. Υπάρχει το score καθώς και σχετικός ήχος για να ενσωματώνει τον παίκτη στο θέμα του παιχνιδιού.
